# FolderOrganizer

FolderOrganizer は、  
**フォルダ／ファイル名を安全に整理するための macOS アプリ**です。

単なる一括リネームではなく、

- 事前に差分を確認できる
- 実行結果が明確に分かる
- いつでも元に戻せる

という「安心して使える整理体験」を重視して設計されています。

---

## 🎯 目的

- 雑多になったフォルダ構成を整理したい
- 自動リネームはしたいが、誤変更は避けたい
- 人の判断を残しつつ、作業を効率化したい

FolderOrganizer は  
**「自動化」と「人の確認」の中間**を狙ったツールです。

---

## ✨ 主な機能（v0.1）

### Preview
- フォルダ配下をスキャンし、リネーム候補を一覧表示
- original / normalized の差分を文字単位で確認可能
- スペース（半角・全角）の可視化に対応

### Apply
- 変更内容を確認した上でリネームを実行
- 変更なしの項目は skipped として明示
- 実行結果を一覧で確認可能

### Undo
- Apply 実行前の状態へ確実にロールバック
- ファイル移動履歴に基づいた安全な Undo を実装

---

## 🧩 設計思想

- **壊れないことを最優先**
- 自動化しすぎない（必ず確認できる）
- 状態を曖昧にしない（skipped / changed / failed を区別）

UI / Domain / Service を分離した構成で、
将来的な拡張を前提に設計されています。
